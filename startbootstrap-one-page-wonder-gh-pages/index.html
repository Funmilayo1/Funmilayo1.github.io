Environmental Friendly Clothes Near You

About The Creators...

This project was created by Myah Mitchell, Adeshina Funmilayo, Farida Larry, and Asanti Aoston. Farida Larry is the leader. She came up with the original idea for this project. The idea was to create a website that allows people to be more aware of eco friendly clothing near them. While also shedding light on the brands that use sweatshops.
